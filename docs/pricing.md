# AI Code Assist - 料金体系

## 料金プラン

### フリープラン
- 料金：無料
- 利用制限：
  - 1日100回までのAIコード生成
  - 基本的なコードレビュー機能
  - コミュニティサポート
- 対象：個人開発者、学習者

### プロプラン
- 料金：月額2,980円（税込）
- 機能：
  - 無制限のAIコード生成
  - 高度なコードレビュー機能
  - プライオリティサポート
  - チーム管理機能
- 対象：プロフェッショナル開発者

### ビジネスプラン
- 料金：月額9,800円（税込）
- 機能：
  - プロプランの全機能
  - カスタマイズ可能なAIモデル
  - 専任サポート
  - セキュリティ監査
  - SLA保証
- 対象：企業、開発チーム

### エンタープライズプラン
- 料金：要相談
- 機能：
  - ビジネスプランの全機能
  - カスタム開発対応
  - オンプレミスデプロイ
  - 24/7専任サポート
  - セキュリティ監査
- 対象：大規模企業

## 支払い方法
- クレジットカード決済
- 銀行振込
- 請求書支払い（ビジネスプラン以上）

## 契約期間
- 月額契約
- 年額契約（10%割引）
- 3年契約（15%割引）

## 解約条件
- 契約期間終了後、次回更新日の7日前までに解約可能
- 中途解約の場合、残り期間の料金は返金不可

## 無料トライアル
- 全プランで14日間の無料トライアル可能
- クレジットカード情報不要
- トライアル期間中は全機能が利用可能
